# Aamer Majid Bhat — Academic Portfolio

Responsive static academic portfolio for Dr. Aamer Majid Bhat.

Live site: <https://sunilkumardhayal.github.io/aamer/>

## Local preview

Run a static server from this directory:

```powershell
python -m http.server 8080
```

Then open <http://127.0.0.1:8080/>.

## Deployment

The site is designed for GitHub Pages and publishes from the repository root on the `main` branch.
